# To-Do List App

A dynamic To-Do List application built using HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Dynamic task box resizing
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Dynamic completed/total task counter
- Loader progress indicator

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)
